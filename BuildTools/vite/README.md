# Cоздания проекта React + TypeScript с помощью Vite

1. Для создания проекта достаточно ввести в терминал следующую команду:

```bash
npm create vite@latest

```

2. Далее следуя подсказкам в терминале указать:

- Имя проекта: Введите название (например, my-react-ts-app).

- Фреймворк: Выберите React.

- Язык: Выберите TypeScript.
